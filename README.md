# Dendritic Projects Generator

This is a spike of a `yeoman` based generators system for the `dendritic` project

## Usage & Examples

```
npm install -g yo generator-dendritic

yo dendritic SpektakularApp
```

## Development

```
npm install
npm link
npm test
```
