# Redbeard Projects Generator

This is a spike of a `yeoman` based generators system for the `redbeard` project

## Usage & Examples

```
npm install -g yo generator-redbeard

yo redbeard SpektakularApp
```

## Development

```
npm install
npm link
npm test
```
